HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is where the main content is, whereas CSS controls appearance and JS is for scripting.
  • img src is used to display images (don't forget you can link with them too).
  • href is used for links; links can include hypertext, CSS sheets, or script files.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS file needs to be linked to the HTML page by referring to it in the Header.
  • CSS is used more globally to specify a style direction as a template for sites as a whole.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creats a new branch and switches to it.
  • git add -A: stages changes after saved
  • git commit -m "": add a message to the commit
  • git push origin: pushes up all of our changes
  • git pull origin main : pulls down current main branch so that we're up to date

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Javascript reads from the top first.
  • Functions in Javascript work pretty similar to other programming languages.
  • There was a sentence that implied garbage collection, but that needs to be verified.
